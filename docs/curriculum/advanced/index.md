# Advanced Java Theory

This section continues the original Java outline after exceptions. These topics help students write larger programs, reason about performance, and choose better data structures.

## Learning Goals

By the end of this section, students should be able to:

- Read and write files safely.
- Use lambdas, generics, lists, maps, and streams.
- Explain recursion and Big O notation.
- Implement and use linked lists, stacks, queues, search algorithms, trees, heaps, and graphs.
- Choose a data structure based on the problem instead of habit.
