# Java Fundamentals

These lessons cover the Java skills a student needs before taking on WPILib robot code. The section ends at exceptions so students understand control flow, objects, interfaces, data containers, and failure handling before moving into the command-based robot framework.

## Learning Goals

By the end of this section, students should be able to:

- Create and run a simple Java program.
- Use primitive values, strings, arrays, enums, records, and simple objects.
- Break logic into readable methods.
- Model behavior with inheritance, abstract classes, and interfaces.
- Sort primitive and object data.
- Handle errors with exceptions instead of silent failure.
