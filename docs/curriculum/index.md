# Curriculum

This curriculum is organized as a path from basic Java through practical WPILib robot programming and then into the data structures and algorithms that make larger robot projects easier to reason about.

<div class="grid cards" markdown>

-   :lucide-code-xml: **Java Fundamentals**

    ---

    Start here if you are new to Java or need a clean review. This section ends at exceptions, which is the minimum foundation students should have before working deeply in robot code.

-   :lucide-bot: **WPILib**

    ---

    Build the mental model for FRC robot projects: lifecycle methods, constants, subsystems, commands, controllers, command groups, and autonomous routines.

-   :lucide-network: **Advanced Java Theory**

    ---

    Continue the original Java outline after exceptions. This section covers file IO, collections, lambdas, streams, recursion, Big O, and common data structures.

</div>

## Recommended Order

1. Complete **Java Fundamentals** through exceptions.
2. Build small robot programs in **WPILib**.
3. Use **Advanced Java Theory** when robot code starts needing more structure, smarter data handling, or better performance reasoning.

!!! note "Instructor note"
    The sections are split by teaching purpose, not by difficulty alone. WPILib is placed after exceptions because students need enough Java to read robot templates, but they do not need every data structure before writing useful robot code.
