# WPILib

This section turns the Java fundamentals into practical FRC robot code. The goal is not to memorize every WPILib class. The goal is to understand the shape of a command-based robot project well enough to read examples, debug behavior, and safely add new mechanisms.

## Learning Goals

By the end of this section, students should be able to:

- Create and run a WPILib Java robot project.
- Explain robot lifecycle methods.
- Store ports, gains, and measurements in constants.
- Model mechanisms as subsystems.
- Control motors safely.
- Write commands and bind them to controllers.
- Combine commands into autonomous routines.
